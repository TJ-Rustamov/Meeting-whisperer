📋 WHAT EACH FILE DOES
═══════════════════════════════════════════════════════════════════════════════

app_launcher.py
  • Entry point when user runs the application
  • Starts FastAPI backend server automatically
  • Serves React frontend automatically
  • Opens browser to http://localhost:5173
  • Manages process lifecycle (start, stop, restart)
  • Stores user data in: %APPDATA%\MeetingWhisperer\...\data\

app_launcher.spec
  • Tells PyInstaller what to bundle:
    - Your backend code (app/)
    - Your frontend (dist/)
    - Your models (models/)
    - All Python packages (torch, fastapi, etc.)
  • Creates standalone executable that doesn't rely on system Python

installer.iss
  • Professional Windows installer configuration
  • Creates Setup.exe that users run
  • Shows installer wizard
  • Creates Start Menu shortcuts
  • Allows uninstall from Control Panel
  • Runs .bat to create installation media

build.bat / quickbuild.bat
  • Automate the entire build process
  • Step 1: Install dependencies
  • Step 2: Build frontend (npm run build)
  • Step 3: Create executable (PyInstaller)
  • Step 4: Create installer (if Inno Setup installed)

check_requirements.bat
  • Checks if prerequisite tools are installed
  • Verifies: Python, Node.js, npm, Git, FFmpeg
  • Checks: disk space, ports
  • Helps diagnose build issues

🔧 CUSTOMIZATION OPTIONS
═══════════════════════════════════════════════════════════════════════════════

Easy customizations you can make:

1. CHANGE APP NAME:
   Edit in app_launcher.py:
   APP_NAME = "My App"

2. CHANGE PORTS:
   Edit in app_launcher.py:
   BACKEND_PORT = 8000  (change to 8001, 8002, etc.)
   FRONTEND_PORT = 5173 (change to 5174, 5175, etc.)

3. ENABLE GPU:
   Edit in app_launcher.py:
   env.update({'WHISPER_DEVICE': 'cuda'})

4. CUSTOMIZE SETTINGS:
   Edit config.example.ini and create .env files

5. ADD CUSTOM ICON:
   Place 256x256 PNG image as launcher/assets/app.ico
   Rebuild the app

6. CHANGE VERSION:
   Edit #define MyAppVersion "1.0.0" in installer.iss

🎓 HOW IT ALL WORKS
═══════════════════════════════════════════════════════════════════════════════

1. USER PERSPECTIVE:
   - Downloads MeetingWhisperer-Setup-1.0.0.exe
   - Clicks installer like any Windows app
   - Installer extracts all files to Program Files
   - Creates Start Menu shortcut
   - User clicks shortcut → App opens in browser
   - App works exactly like Docker version

2. UNDER THE HOOD:
   - PyInstaller bundles Python + all code + dependencies
   - Inno Setup creates professional Windows installer
   - app_launcher.py starts backend & frontend automatically
   - Data stored in user AppData (survives reinstalls)
   - No Docker required, no system Python needed

3. DISTRIBUTION:
   - Share installer .exe file with users
   - Users install like any Windows application
   - Updates: New installer preserves data in AppData
   - Uninstall: Removes app but keeps user data option
